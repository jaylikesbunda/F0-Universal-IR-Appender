Universal IR Appender v1.1

How to Use

  1. Select the folder containing your .ir files.
  2. Choose the appropriate device type from the dropdown.
  3. Click "Append to Universal IR File" to start processing.
  4. After processing, view the summary and export or copy the results as needed.

Note: Ensure your .ir files are named in the format Brand_Model.ir for optimal device identification.

Device Naming Guidelines

The application attempts to extract the device's Brand and Model from the file content. If not found, it expects the filename to follow the Brand_Model.ir format. Fallback: If both methods fail, the entire filename (without extension) will be used as the device identifier.
